Work

Everything you see here is fully customizable. Images can be your own or stock ones. There's no limit to the beauty of Google Fonts, although I wouldn't use more than two, maybe three. And you can have as many extra sections or columns as you want. Same goes for adding as many content pages as you need.

Please note these, as my own website, are static designs. Meaning I will have to learn a bit more to make them look perfect on a mobile device. After all, this is not a sales pitch, I'm only showing the current state of my skills.

Personal Style Website

This was my first HTML+CSS excersice and it's the basis of the website you're visiting right now. Since I had so much questions and subsequent guidance while doing it, I know it pretty well.

My favorite thing while coding this was practicing how to add images using CSS instead of HTML, which allows for text to be superimposed on the pictures.

I'd recommed something like this if you want a neat, easy to read website with minor navigation, and a professional look.
